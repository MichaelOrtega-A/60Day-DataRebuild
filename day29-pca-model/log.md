# 🧠 Day 29: Model Training on PCA-Reduced Features

### ✅ Objectives:
- Train Random Forest on PCA-reduced features
- Compare performance to original feature space

### 🔍 Key Results:
- Original Features (8): **Accuracy = 81.0%**
- PCA-Reduced (7 components): **Accuracy = 78.2%**
- Small drop in accuracy (~3%) after PCA
- Confusion matrix and classification report show comparable balance

### 🧠 Insight:
- PCA helps simplify feature space with minimal loss in performance
- Valuable when working with high-dimensional or noisy datasets
