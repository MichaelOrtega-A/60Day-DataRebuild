# Day 13 Log – Model Comparison

Today I compared five machine learning models using 5-fold cross-validation on my engineered Titanic dataset:

- Logistic Regression: 79.35%
- Random Forest: 80.36%
- Decision Tree: 77.45%
- K-Nearest Neighbors: 80.93%s
- **Support Vector Machine: 83.05%**

SVM came out on top with the highest average accuracy, especially after scaling.  
I’m starting to see how different models behave and what they need to perform well.
