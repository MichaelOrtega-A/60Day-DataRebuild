# 🧠 Day 31: Model Comparison

### ✅ Objective:
Compare multiple machine learning models using consistent preprocessing and 5-fold cross-validation.

### 📊 Models Compared:
- Logistic Regression
- Random Forest
- K-Nearest Neighbors
- Support Vector Machine

### 🔍 Results:

| Model | Mean Accuracy | Std Dev |
|-------|---------------|---------|
| Support Vector Machine | 83.05% | 0.0198 |
| Random Forest | 81.93% | 0.0181 |
| K-Nearest Neighbors | 81.26% | 0.0165 |
| Logistic Regression | 79.35% | 0.0205 |

### 🧠 Insight:
- **SVM outperformed all others** with 83.05% accuracy and consistent performance.
- **Random Forest** is a close second and easier to tune.
- This comparison sets the stage for future hyperparameter optimization.
