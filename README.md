# 60Day-DataRebuild 📊

A 60-day data science self-rebuild plan — from scratch to job-ready.

---

## 🔥 Purpose

After stepping away from tech and working physical jobs, I’m committing 60 days to rebuild my data skills, portfolio, and confidence. This repo documents the entire journey.

---

## 💡 Goals

- Re-learn Python, Pandas, SQL, and Data Visualization
- Complete 2–3 real portfolio projects
- Build resume-ready artifacts hosted on GitHub
- Apply to 20+ data jobs by the end of this plan

---

## 📁 Structure

- `day01-python-warmups/`: Daily practice tasks and notes
- `projects/`: Cleaned-up notebooks and final projects
- `week2-sql-drills/`: SQL queries and practice
- `README.md`: Plan, logs, and reflections
